# Clustering and Similarity Analysis of IR Spectra

This repository contains a Jupyter Notebook for generating, processing, and comparing **infrared (IR) vibrational spectra** using numerical broadening techniques and similarity metrics. The workflow focuses on constructing vibrational density of states (VDOS) spectra and evaluating similarities between multiple spectra.

---

## Features

- Gaussian and Lorentzian spectral broadening
- Vibrational Density of States (VDOS) construction
- Multiple spectral similarity metrics:
  - Euclidean similarity
  - Cosine similarity
  - Pearson correlation
- Pairwise similarity matrix computation
- Heatmap visualization for clustering analysis
